NoSQL databáze

NoSQL databáze

(Non-relational, nebo "Not Only SQL") jsou nerelační databáze ukládající data v různých formátech (dokumenty, klíč-hodnota, grafy, sloupce) místo tradičních tabulek, což nabízí větší flexibilitu (dynamické schéma), vysokou škálovatelnost (horizontální škálování na více serverů) a efektivitu pro velké objemy nestrukturovaných dat, jako jsou obrázky, videa a texty, a jsou ideální pro moderní webové aplikace a Big Data, i když často obětují úplnou konzistenci {1, 2, 4, 5, 9}.

Jak fungují

  • Flexibilní schéma: Různé záznamy (dokumenty) v databázi nemusí mít stejnou strukturu, což umožňuje snadné přidávání nových polí bez změn celé databáze, na rozdíl od pevného schématu SQL {3, 4}.
  • Různé datové modely: Nepoužívají tabulky s řádky a sloupci. Místo toho využívají modely jako:
    • Dokumentové: Ukládají data ve formátu jako JSON, BSON (např. MongoDB, Elasticsearch) {1, 8}.
    • Klíč-hodnota: Ukládají data jako jednoduché páry klíč-hodnota (např. Redis) {8}.
    • Sloupcové: Ukládají data ve sloupcových rodinách (např. Cassandra) {8}.
    • Grafové: Ukládají data jako uzly a hrany (vztahy), skvělé pro propojování dat (např. Neo4j) {8, 10}.
  • Horizontální škálování: Snadno zvládají nárůst dat a provozu rozložením dat na více levnějších serverů (shardů) {7, 13}.

Výhody

  • Flexibilita: Ideální pro rychle se měnící nebo nestrukturovaná data {1, 9}.
  • Škálovatelnost: Perfektní pro Big Data a aplikace s vysokou návštěvností {1, 4, 7}.
  • Výkon: Rychlé čtení a zápis pro specifické typy dotazů. 

Nevýhody

  • Méně konzistence: Často upřednostňují dostupnost před okamžitou konzistencí (tzv. eventual consistency) {2, 13}.
  • Složitější dotazy: Složité dotazy vyžadující "joins" (spojování) dat z různých kolekcí/dokumentů mohou být náročnější než v SQL databázích. 

Kdy je použít

  • Pro Big Data (velké objemy dat), jako jsou uživatelské profily, logy, data ze sociálních sítí, e-commerce katalogy {1, 4, 9}.
  • Pro aplikace vyžadující vysokou dostupnost a rychlé škálování, jako jsou real-time analytika, IoT a mobilní aplikace.
