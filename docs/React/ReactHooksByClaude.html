Introduction to React Hooks

React Hooks, introduced in React 16.8, enable functional components to use state, lifecycle, and other React features without relying on class components.

Key Benefits

  • Eliminate class component requirements for state management
  • Improve code readability with functional programming style
  • Enable cleaner, more maintainable modern React applications

1. State Hooks

useState

Used to declare state variables in functional components, allowing you to read and modify state.

Syntax:

const [state, setState] = useState(initialState);

Parameter

Description

state

Current state value

setState

Function to update state

initialState

Initial value (primitive, object, or array)

Example – Counter:

import React, { useState } from "react";

function App() {
    const [count, setCount] = useState(0);
    const increment = () => setCount(count + 1);
    const decrement = () => setCount(count - 1);

    return (
        <div>
            <h1>Count: {count}</h1>
            <button onClick={increment}>Increment</button>
            <button onClick={decrement}>Decrement</button>
        </div>
    );
}

export default App;

useReducer

Advanced state management for complex logic involving multiple sub-values or intricate state transitions.

Syntax:

const [state, dispatch] = useReducer(reducer, initialState);

Example:

import React, { useReducer } from "react";

function reducer(count, action) {
    if (action === "increment") return count + 1;
    if (action === "decrement") return count - 1;
    return count;
}

function App() {
    const [count, dispatch] = useReducer(reducer, 0);

    return (
        <div>
            <h1>Count: {count}</h1>
            <button onClick={() => dispatch("increment")}>Increment</button>
            <button onClick={() => dispatch("decrement")}>Decrement</button>
        </div>
    );
}

export default App;

2. Context Hooks

useContext

Allows functional components to directly access React Context API values without prop drilling.

Syntax:

const contextValue = useContext(MyContext);

Example – Theme context:

import React, { createContext, useContext, useState } from "react";

const ThemeContext = createContext();

function App() {
    const [theme, setTheme] = useState("light");

    return (
        <ThemeContext.Provider value={theme}>
            <div>
                <h1>Current Theme: {theme}</h1>
                <button onClick={() => setTheme(t => t === "light" ? "dark" : "light")}>
                    Toggle Theme
                </button>
                <ThemeDisplay />
            </div>
        </ThemeContext.Provider>
    );
}

function ThemeDisplay() {
    const theme = useContext(ThemeContext);
    return <h2>Theme from Context: {theme}</h2>;
}

export default App;

3. Effect Hooks

useEffect

Handles common side effects like data fetching, subscriptions, and DOM manipulation. Replaces componentDidMount and componentDidUpdate.

Syntax:

useEffect(() => {
    // Side effect logic here
    return () => { /* cleanup */ };
}, [dependencies]);

Example – document title:

import React, { useState, useEffect } from "react";

function App() {
    const [count, setCount] = useState(0);

    useEffect(() => {
        document.title = `Count: ${count}`;
        return () => { document.title = "React App"; };
    }, [count]);

    return (
        <div>
            <h1>Count: {count}</h1>
            <button onClick={() => setCount(count + 1)}>Increment</button>
        </div>
    );
}

export default App;

useLayoutEffect

Runs synchronously after DOM updates, before the browser paints. Useful for measuring or modifying layout.

useLayoutEffect(() => {
    // Manipulate layout or measure DOM elements
}, [dependencies]);

4. Performance Hooks

useMemo

Caches computed results, recalculating only when dependencies change. Prevents unnecessary recalculations on every render.

Syntax:

const memoizedValue = useMemo(() => computeExpensiveValue(a, b), [a, b]);

Example:

import React, { useState, useMemo } from "react";

function App() {
    const [count, setCount] = useState(0);
    const [text, setText] = useState("");

    const expensiveCalculation = useMemo(() => {
        console.log("Expensive calculation...");
        return count * 2;
    }, [count]);

    return (
        <div>
            <h1>Count: {count}</h1>
            <h2>Result: {expensiveCalculation}</h2>
            <button onClick={() => setCount(count + 1)}>Increment Count</button>
            <input value={text} onChange={e => setText(e.target.value)} />
        </div>
    );
}

export default App;

useCallback

Memoizes function references so they are only recreated when dependencies change. Prevents unnecessary re-renders of child components that receive functions as props.

Syntax:

const memoizedCallback = useCallback(() => { doSomething(a, b); }, [a, b]);

5. Ref Hooks

useRef

Stores mutable values or accesses DOM elements without triggering re-renders.

Syntax:

const refContainer = useRef(initialValue);

Example – Accessing DOM element:

import React, { useState, useRef } from "react";

function App() {
    const countRef = useRef(0);
    const [forceRender, setForceRender] = useState(false);

    const increment = () => {
        countRef.current += 1;
        setForceRender(!forceRender);
    };

    return (
        <div>
            <h1>Count: {countRef.current}</h1>
            <button onClick={increment}>Increment</button>
        </div>
    );
}

export default App;

6. Custom Hooks

User-defined functions starting with "use" that encapsulate reusable logic. Enhance code reusability without altering component hierarchy.

Example – useWidth:

// useWidth.js
import { useState, useEffect } from "react";

function useWidth() {
    const [width, setWidth] = useState(window.innerWidth);

    useEffect(() => {
        const handleResize = () => setWidth(window.innerWidth);
        window.addEventListener("resize", handleResize);
        return () => window.removeEventListener("resize", handleResize);
    }, []);

    return width;
}

export default useWidth;

Using the custom hook:

import React from "react";
import useWidth from "./useWidth";

function App() {
    const width = useWidth();
    return <h1>Window Width: {width}px</h1>;
}

export default App;

What is a Hook?

Hooks are functions that let you 'hook into' React state and lifecycle features from functional components. They do not work inside class components.

Hook Rules

  • Hooks can only be called inside React function components.
  • Hooks can only be called at the top level of a component.
  • Hooks cannot be conditional.
Example
import { useState } from 'react';
import { createRoot } from 'react-dom/client';

function FavoriteColor() {
  const [color, setColor] = useState("red");

  return (
    <>
      <h1>My favorite color is {color}!</h1>
      <button type="button" onClick={() => setColor("blue")}>Blue</button>
      <button type="button" onClick={() => setColor("red")}>Red</button>
      <button type="button" onClick={() => setColor("pink")}>Pink</button>
      <button type="button" onClick={() => setColor("green")}>Green</button>
    </>
  );
}

createRoot(document.getElementById('root')).render(<FavoriteColor />);

useState

The React useState Hook allows you to track state in a function component. State generally refers to data or properties that need to be tracked in an application.

Syntax:

const [currentValue, setCurrentValue] = useState(initialValue);

Reading and Updating State

Example – multiple state variables
function MyCar() {
  const [brand, setBrand] = useState("Ford");
  const [model, setModel] = useState("Mustang");
  const [year, setYear] = useState("1964");
  const [color, setColor] = useState("red");

  return (
    <>
      <h1>My {brand}</h1>
      <p>It is a {color} {model} from {year}.</p>
    </>
  )
}

Updating Objects in State

When state is updated the entire state gets replaced. Use the spread operator to update only the changed property:

Example – spread operator
function MyCar() {
  const [car, setCar] = useState({
    brand: "Ford",
    model: "Mustang",
    year: "1964",
    color: "red"
  });

  const updateColor = () => {
    setCar(previousState => {
      return { ...previousState, color: "blue" }
    });
  }

  return (
    <>
      <h1>My {car.brand}</h1>
      <p>It is a {car.color} {car.model} from {car.year}.</p>
      <button type="button" onClick={updateColor}>Blue</button>
    </>
  )
}

useEffect

The useEffect Hook allows you to perform side effects in your components. Some examples of side effects are: fetching data, directly updating the DOM, and timers.

Syntax:

useEffect(<function>, <dependency>)

Dependency Array Patterns

Pattern

When it runs

useEffect(() => { })

Runs on every render

useEffect(() => { }, [])

Runs only on the first render

useEffect(() => { }, [prop, state])

Runs on first render and when prop/state changes

Example – Timer with cleanup
import { useState, useEffect } from 'react';
import { createRoot } from 'react-dom/client';

function Timer() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    let timer = setTimeout(() => {
      setCount((count) => count + 1);
    }, 1000);

    return () => clearTimeout(timer);
  }, []);

  return <h1>I've rendered {count} times!</h1>;
}

createRoot(document.getElementById('root')).render(<Timer />);
Example – Dependent effect
function Counter() {
  const [count, setCount] = useState(0);
  const [calculation, setCalculation] = useState(0);

  useEffect(() => {
    setCalculation(() => count * 2);
  }, [count]);

  return (
    <>
      <p>Count: {count}</p>
      <button onClick={() => setCount((c) => c + 1)}>+</button>
      <p>Calculation: {calculation}</p>
    </>
  );
}

Create and Use Context – 3 Steps

  • Step 3: Call useContext(MyContext) in any child component

useRef

The useRef Hook allows you to persist values between renders without triggering a re-render, and to access DOM elements directly.

Example – track render count (no re-render)
import { useState, useRef, useEffect } from 'react';

function App() {
  const [inputValue, setInputValue] = useState("");
  const count = useRef(0);

  useEffect(() => {
    count.current = count.current + 1;
  });

  return (
    <>
      <input
        type="text"
        value={inputValue}
        onChange={(e) => setInputValue(e.target.value)}
      />
      <h1>Render Count: {count.current}</h1>
    </>
  );
}
Example – focus a DOM element
import { useRef } from 'react';

function App() {
  const inputElement = useRef();

  const focusInput = () => {
    inputElement.current.focus();
  };

  return (
    <>
      <input type="text" ref={inputElement} />
      <button onClick={focusInput}>Focus Input</button>
    </>
  );
}
Example – track previous state value
import { useRef, useState, useEffect } from 'react';

function App() {
  const [inputValue, setInputValue] = useState("");
  const previousInputValue = useRef("");

  useEffect(() => {
    previousInputValue.current = inputValue;
  }, [inputValue]);

  return (
    <>
      <input
        type="text"
        value={inputValue}
        onChange={(e) => setInputValue(e.target.value)}
      />
      <h2>Current Value: {inputValue}</h2>
      <h2>Previous Value: {previousInputValue.current}</h2>
    </>
  );
}

useReducer

The useReducer Hook is similar to the useState Hook. It allows for custom state logic and is useful when you find yourself keeping track of multiple pieces of state that rely on complex logic.

Syntax:

useReducer(reducer, initialState, init)

Parameter

Description

reducer

Function containing custom state logic

initialState

Initial state value (simple value or object)

init

Optional function for lazy state initialisation

Example – scoreboard
import { useReducer } from 'react';

const initialScore = [
  { id: 1, score: 0, name: "John" },
  { id: 2, score: 0, name: "Sally" },
];

const reducer = (state, action) => {
  switch (action.type) {
    case "INCREASE":
      return state.map((player) => {
        if (player.id === action.id) {
          return { ...player, score: player.score + 1 };
        }
        return player;
      });
    default:
      return state;
  }
};

function Score() {
  const [score, dispatch] = useReducer(reducer, initialScore);

  return (
    <>
      {score.map((player) => (
        <div key={player.id}>
          <label>
            <input
              type="button"
              onClick={() => dispatch({ type: "INCREASE", id: player.id })}
              value={player.name}
            />
            {player.score}
          </label>
        </div>
      ))}
    </>
  );
}

useCallback

The useCallback Hook returns a memoized callback function. This allows you to isolate resource-intensive functions from re-running on every render. Note: useCallback returns a memoized function, useMemo returns a memoized value.

Syntax:

useCallback(callback, dependencies)
Example – with useCallback (only clicked button re-renders)
import React, { useState, useCallback } from 'react';

const Button = React.memo(({ onClick, text }) => {
  console.log(`${text} button rendered`);
  return <button onClick={onClick}>{text}</button>;
});

function WithCallbackExample() {
  const [count1, setCount1] = useState(0);
  const [count2, setCount2] = useState(0);

  const handleClick1 = useCallback(() => {
    setCount1(count1 + 1);
  }, [count1]);

  const handleClick2 = useCallback(() => {
    setCount2(count2 + 1);
  }, [count2]);

  return (
    <div>
      <p>Count 1: {count1}</p>
      <p>Count 2: {count2}</p>
      <Button onClick={handleClick1} text="Button 1" />
      <Button onClick={handleClick2} text="Button 2" />
    </div>
  );
}

useMemo

The useMemo Hook returns a memoized value. It only runs when one of its dependencies updates, improving performance for expensive calculations.

Syntax:

useMemo(() => computeExpensiveValue(a, b), [a, b])
Example – expensive calculation with useMemo
import { useState, useMemo } from 'react';

const expensiveCalculation = (num) => {
  console.log("Calculating...");
  for (let i = 0; i < 1000000000; i++) { num += 1; }
  return num;
};

const App = () => {
  const [count, setCount] = useState(0);
  const [todos, setTodos] = useState([]);

  // Only recalculates when count changes, not when todos change
  const calculation = useMemo(() => expensiveCalculation(count), [count]);

  return (
    <div>
      <div>
        <h2>My Todos</h2>
        {todos.map((todo, index) => <p key={index}>{todo}</p>)}
        <button onClick={() => setTodos((t) => [...t, "New Todo"])}>Add Todo</button>
      </div>
      <hr />
      <div>
        Count: {count}
        <button onClick={() => setCount((c) => c + 1)}>+</button>
        <h2>Expensive Calculation: {calculation}</h2>
      </div>
    </div>
  );
};

Custom Hooks

Hooks are reusable functions. When you have component logic that needs to be used by multiple components, we can extract that logic to a custom Hook. Custom Hooks start with "use".

useFetch.js – custom hook
import { useState, useEffect } from "react";

const useFetch = (url) => {
  const [data, setData] = useState(null);

  useEffect(() => {
    fetch(url)
      .then((res) => res.json())
      .then((data) => setData(data));
  }, [url]);

  return [data];
};

export default useFetch;
index.js – using the custom hook
import { createRoot } from 'react-dom/client';
import useFetch from "./useFetch";

const Home = () => {
  const [data] = useFetch("https://jsonplaceholder.typicode.com/todos");

  return (
    <>
      {data &&
        data.map((item) => {
          return <p key={item.id}>{item.title}</p>;
        })}
    </>
  );
};

createRoot(document.getElementById('root')).render(<Home />);

useState is a React Hook that lets you add a state variable to your component.

Parameters & Returns

Description

The initial value. If a function is passed it is treated as an initializer (called once only).

Returns [state, set]

Updater Function (avoid stale state)

Wrong – multiple calls in same event use stale value:

// All three calls read the same stale age!
function handleClick() {
  setAge(age + 1); // setAge(42 + 1)
  setAge(age + 1); // setAge(42 + 1) — still 42!
  setAge(age + 1); // result is 43, not 45
}

Correct – use updater function:

function handleClick() {
  setAge(a => a + 1); // 42 => 43
  setAge(a => a + 1); // 43 => 44
  setAge(a => a + 1); // 44 => 45
}

Updating Objects and Arrays

Always replace, never mutate:

// Wrong: mutation
form.firstName = 'Taylor';
setForm(form);

// Correct: replacement
setForm({
  ...form,
  firstName: 'Taylor'
});

// Array: add
setTodos([...todos, { id: nextId++, title: title, done: false }]);

// Array: update
setTodos(todos.map(t => t.id === nextTodo.id ? nextTodo : t));

// Array: delete
setTodos(todos.filter(t => t.id !== todoId));

useReducer

useReducer is a React Hook that lets you add a reducer to your component. Useful for complex state with multiple sub-values or interdependent updates.

const [state, dispatch] = useReducer(reducer, initialArg, init?)
import { useReducer } from 'react';

function reducer(state, action) {
  switch (action.type) {
    case 'incremented_age':
      return { name: state.name, age: state.age + 1 };
    case 'changed_name':
      return { name: action.nextName, age: state.age };
    default:
      throw Error('Unknown action: ' + action.type);
  }
}

function MyComponent() {
  const [state, dispatch] = useReducer(reducer, { name: 'Taylor', age: 42 });

  return (
    <>
      <input
        value={state.name}
        onChange={e => dispatch({ type: 'changed_name', nextName: e.target.value })}
      />
      <button onClick={() => dispatch({ type: 'incremented_age' })}>
        Increment age
      </button>
      <p>Hello, {state.name}. You are {state.age}.</p>
    </>
  );
}

Managing Arrays with useReducer

function tasksReducer(tasks, action) {
  switch (action.type) {
    case 'added':
      return [...tasks, { id: action.id, text: action.text, done: false }];
    case 'changed':
      return tasks.map(t => t.id === action.task.id ? action.task : t);
    case 'deleted':
      return tasks.filter(t => t.id !== action.id);
    default:
      throw Error('Unknown action: ' + action.type);
  }
}

Stopwatch – combining ref and state:

import { useState, useRef } from 'react';

export default function Stopwatch() {
  const [startTime, setStartTime] = useState(null);
  const [now, setNow] = useState(null);
  const intervalRef = useRef(null);

  function handleStart() {
    setStartTime(Date.now());
    setNow(Date.now());
    clearInterval(intervalRef.current);
    intervalRef.current = setInterval(() => {
      setNow(Date.now());
    }, 10);
  }

  function handleStop() {
    clearInterval(intervalRef.current);
  }

  const secondsPassed = startTime && now ? (now - startTime) / 1000 : 0;

  return (
    <>
      <h1>Time passed: {secondsPassed.toFixed(3)}</h1>
      <button onClick={handleStart}>Start</button>
      <button onClick={handleStop}>Stop</button>
    </>
  );
}

useEffect

useEffect is a React Hook that lets you synchronize a component with an external system.

useEffect(setup, dependencies?)

Parameter

Description

setup

Function with effect logic. May return a cleanup function.

dependencies?

Array of reactive values. Effect re-runs when any change. Omit to run after every render.

Connecting to External Systems

import { useState, useEffect } from 'react';

function ChatRoom({ roomId }) {
  const [serverUrl, setServerUrl] = useState('https://localhost:1234');

  useEffect(() => {
    const connection = createConnection(serverUrl, roomId);
    connection.connect();
    return () => {
      connection.disconnect();
    };
  }, [serverUrl, roomId]);

  return <h1>Welcome to the {roomId} room!</h1>;
}

useMemo

useMemo is a React Hook that lets you cache the result of a calculation between re-renders.

const cachedValue = useMemo(calculateValue, dependencies)

Skipping Expensive Recalculations

import { useMemo } from 'react';

function TodoList({ todos, tab, theme }) {
  const visibleTodos = useMemo(
    () => filterTodos(todos, tab),
    [todos, tab]
  );

  return (
    <div className={theme}>
      <ul>
        {visibleTodos.map(todo => (
          <li key={todo.id}>{todo.text}</li>
        ))}
      </ul>
    </div>
  );
}

Skipping Re-renders with memo

import { useMemo, memo } from 'react';

const List = memo(function List({ items }) {
  return items.map(item => <li key={item.id}>{item.text}</li>);
});

export default function TodoList({ todos, tab, theme }) {
  // Without useMemo, List re-renders on every parent render
  const visibleTodos = useMemo(
    () => filterTodos(todos, tab),
    [todos, tab]
  );

  return (
    <div className={theme}>
      <List items={visibleTodos} />
    </div>
  );
}

  return (

  );
}

Custom Hooks

You can create your own Hooks to share stateful logic between components. A custom Hook is a JavaScript function whose name starts with "use" and that may call other Hooks.

// useChatRoom.js
function useChatRoom({ serverUrl, roomId }) {
  useEffect(() => {
    const connection = createConnection(serverUrl, roomId);
    connection.connect();
    return () => connection.disconnect();
  }, [roomId, serverUrl]);
}

// Usage in a component
function ChatRoom({ roomId }) {
  const [serverUrl, setServerUrl] = useState('https://localhost:1234');
  useChatRoom({ roomId, serverUrl });
  // ...
}
// useWindowSize.js
import { useState, useEffect } from 'react';

function useWindowSize() {
  const [size, setSize] = useState({
    width: window.innerWidth,
    height: window.innerHeight,
  });

  useEffect(() => {
    function handleResize() {
      setSize({ width: window.innerWidth, height: window.innerHeight });
    }
    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  return size;
}

// Usage
function App() {
  const { width, height } = useWindowSize();
  return <p>Window: {width} x {height}</p>;
}
